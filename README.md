GitHub Actions: Matrix Build Demo

This repository contains a GitHub Actions workflow to demonstrate a multi-platform matrix build.

The workflow (.github/workflows/matrix-build.yml) is configured to:

Run in parallel on ubuntu-latest, macos-latest, and windows-latest.

Generate a unique text file for each operating system.

Upload each file as a separate build artifact, named with the prefix build-040654e-.
